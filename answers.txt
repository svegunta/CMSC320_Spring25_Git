Q1: Why might we consider using Git?

Answer: Git is not only an extremely useful tool for securely storing code and tracking version control, but it is also the industry standard for doing so and working collaboratively.

Q2: What field could data science be applied to? Give an Example.

Answer: Data science can be applied to all fields. An example in engineering is using data science to track performance variation in the same aircraft part across hundreds of aircraft, to determine if any changes are needed.

Q3: What is the difference between Merging and Rebasing?

Answer: The difference between merging and rebasing is that merging preserves the history of both branches and joins them together, while rebasing overwrites one branch with another.
